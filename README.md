# CSS Gradient Generator

Веб-приложение для создания и настройки CSS градиентов с поддержкой анимации и экспорта в различные форматы.

## 🌟 Возможности

### Типы градиентов

- **Linear** — линейный градиент с настраиваемым углом направления (0°-360°)
- **Radial** — радиальный градиент (круг или эллипс)
- **Conic** — конический градиент

### Настройка цветов

- Добавление до 8 цветовых точек
- Настройка позиции каждого цвета (0-100%)
- Выбор цвета через палитру
- Удаление цветовых точек (минимум 2)
- 12 готовых цветовых пресетов

### Анимация градиентов

- **Shift** — плавный сдвиг градиента
- **Rotate** — вращение градиента
- **Hue** — циклическая смена оттенков
- Настройка скорости анимации (1-20 секунд)
- Выбор функции плавности (ease, linear, ease-in, ease-out)

### Экспорт

- 📋 Копирование CSS в буфер обмена
- 💾 CSS файл (.css)
- 🖼️ PNG изображение (1920x1080)
- 🎨 Конфигурация для Tailwind CSS
- 📄 Готовые CSS классы (фон, текст,边框, кнопка)

## 🚀 Использование

1. Откройте `index.html` в браузере
2. Выберите тип градиента (Linear/Radial/Conic)
3. Настройте направление с помощью слайдера или кнопок
4. Добавьте или измените цвета
5. При необходимости включите анимацию
6. Экспортируйте результат в нужном формате
